前语：sublime和markdown语法

sublime安装软件

    1:搜索,ctrl+shift+p打开包管理工具，输入pcip搜索应用安装即可
    2:具体的配置在perferences的package.setting的工具
    3:Key Bind就是浏览器的预览具体配置

具体文章来自 https://www.jianshu.com/p/1e402922ee32/

实用的markdown语法

  1. #表示一级标题 ##二级标题

  2. 表示引用的内容

  3. ！ 图片为 【】（）链接为图片

  4. * 表示斜体 * 表示粗体 \** 分割线 两端都有的**

  5. 列表的显示只需要在文字前加上 - 或 * 即可变为无序列表

  6. 有序列表则直接在文字前加1. 2. 3. 符号要和文字之间加上一个字符的空格

前端摄影师码农

今天是2017学期的倒数5天，有点迷茫,下面分享下我对于光圈,曝光度,亮度大小各个属性值的理解

光圈大小

光圈越大，进光亮越大，景深越淡，实用于拍摄人像背景虚化，人物处理的效果;
光圈越小,进光亮越小,适用于拍摄景色,天空，远方，效果会太好;

快门

快门开启的时间越长就代表快门速度越慢，进光量就越多,反之则快门越快，进光量就越少
快门直接影响照片的亮度
不同的快门速度适用不同的场景

感光度

主要指照片的亮度 数值越低照片越暗，数值越高照片越亮，噪点就越多
尽量ISO调整在100-400之间,
总的来说高的曝光度会带来画质的损失

三者之间的关系

这三种数值是怎么组合的呢
如果想拍摄运动轨迹的效果,就用长快门的组合
如果想拍虚化的效果,那就用大光圈的组合
